# Professional Portfolio Website

A responsive, modern portfolio website to showcase your projects and skills.

## Features

✨ **Features:**
- ✅ Fully responsive design (Mobile, Tablet, Desktop)
- ✅ Sticky navigation menu
- ✅ Hero section with call-to-action
- ✅ About section with skills showcase
- ✅ Projects gallery with descriptions
- ✅ Contact section with multiple channels
- ✅ Social media links
- ✅ Smooth animations and transitions
- ✅ Modern and professional design

## How to Use

1. Download the `index.html` file
2. Open it in your web browser
3. Customize the content with your information
4. Deploy to GitHub Pages or any web host

## Customization

### Change Your Information

Replace these in the HTML:
- `John Doe` - Your name
- `john@example.com` - Your email
- `+1 (234) 567-890` - Your phone
- `San Francisco, California` - Your location
- Add your profile image URL

### Update Projects

Add new project cards by copying the existing project-card div:

```html
<div class="project-card">
    <div class="project-image">🎨</div>
    <div class="project-content">
        <h3>Your Project Title</h3>
        <p class="project-tech">Tech Stack</p>
        <p>Project description...</p>
    </div>
</div>
Customize Colors
Edit these CSS variables:
:root {
    --primary: #2c3e50;      /* Main color */
    --secondary: #3498db;    /* Accent 1 */
    --accent: #e74c3c;       /* Accent 2 */
    --light: #ecf0f1;        /* Background */
}
Deployment Options
Option 1: GitHub Pages (Free)
Create a GitHub repository
Upload index.html
Go to Settings → Pages
Select main branch as source
Your site goes live!
Option 2: Netlify (Free)
Go to netlify.com
Drag and drop your index.html
Done! Site is live
Option 3: Vercel (Free)
Go to vercel.com
Import your GitHub repository
Deploy with one click
